# cosibella products

Aplikacja frontendowa w Vanilla JavaScript do przeglądania produktów pobieranych z API:

```txt
https://s5.cosibella.pl/api/test/products
```

## Stack technologiczny

- HTML5,
- Vanilla JavaScript,
- natywne ES Modules,
- SCSS,
- Vite,
- Sass,
- ESLint,
- Prettier.

## Uruchomienie

Zainstaluj zależności:

```bash
npm install
```

Uruchom aplikację w trybie developerskim:

```bash
npm run dev
```

Komenda uruchamia lokalny serwer Vite:

```txt
http://localhost:5173/
```
